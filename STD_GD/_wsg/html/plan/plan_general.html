구조

영역구분
구분 예약어 설명
머리글 영역 -header 그룹내 유일한 머리글 영역
본문 영역 -container 그룹내 유일한 본문 영역
바닥글 영역 -footer 그룹내 유일한 바닥글 영역
제목 영역 -head 여러개의 제목 영역이 존재할 수 있음.
내용 영역 -cont 여러개의 내용 영역이 존재할 수 있음.
바닥 영역 -foot 여러개의 내용 영역이 존재할 수 있음.
네비게이션 -nav 탐색 역할을 갖는 범위의 메뉴 (예) .tab-nav
메뉴 -menu 탐색 역할이 없는 선택목록 메뉴 (예) .drop-menu, .select-menu
목록 -list ul, ol 에 해당하는 반복되는 목록그룹
항목 -item 리스트 항목
링크 -link 링크 항목
제목 -title 텍스트 항목
텍스트 -text 텍스트 항목
토글버튼 -toggle 토글 기능을 갖는 버튼
열기버튼 -open 열기 기능을 갖는 버튼
닫기버튼 -close 닫기 기능을 갖는 버튼
