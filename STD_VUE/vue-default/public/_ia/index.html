    • 유형 표시방법 - FP : Full Popup, MP : Massge Popup, SP : Select Popup, EP : Etc Popup,
    • 1차 진척관리 - 최초 완료표시 기준으로 날짜를 표시하고, 비고란에 전달사항이 있는경우 표시한다.

    • 3차 최종완료 - 사이트 오픈후에 전체현황을 엑셀로 백업하여 기록을 남기고 비고란은 삭제한다. (최종완료 초기화 작업)
