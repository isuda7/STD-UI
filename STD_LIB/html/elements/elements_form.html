									<span class="file">
										<input type="file" name="sFiles1" id="sFiles1" value="찾아보기" tabindex="-1" aria-hidden="true"/>
										<input type="text" class="input" title="첨부된 파일명" />
										<button type="button" class="btn style-1 btn_file" onclick="fileAttach(this)"><span>파일첨부</span></button>
									</span>
								
									/* Form File */
									.file {position:relative;}
									.file input[type="file"] {position:absolute; width:0; height:0; overflow:hidden;}
									.file input[type="text"] {display:inline-table;}
									.file .btn_file {display:inline-table;}
								
									//파일첨부
									function fileAttach(obj){
										var $this = $(obj).closest('.file');
										$this.find('[type=file]').trigger('click');
										$this.find('[type=file]').not('.is-evented').on('change', function(){
											var value = $(this).val();
											$this.find('[type=text]').val(value);
										}).addClass('is-evented');
									}
설계가이드
  • 묵시적인 방법(label 에 checkbox, radio 포함)은 일부 스크린리더에서 해당 마크업을 제대로 읽지 못하므로
  • 명시적인 방법(input id 와 label for 연결)으로 사용하는 것을 권장합니다.
